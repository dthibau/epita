# Epic: Marketplace

## Description

> Features related to the commercial aspect of the platform: purchasing, reselling, and cart management.

---

## Product Requirements

* **Core Goal:** Provide a secure marketplace for buying and selling game keys.
* **Requirement 1:** Manage the main marketplace for buying game keys natively.
* **Requirement 2:** Develop a user-to-user resell system to trade digital items.
* **Requirement 3:** Handle shopping cart functionality and reliable order tracking.

## Technical Requirements

* **Key Integration:** Integration of Apple and Amazon gift cards for simplified payments.
* **Security/Compliance:** Ensure transactional security across purchases and the resell flow.

---

## Acceptance Criteria

To consider this Epic complete, the following must be true:

1. [ ] The marketplace allows users to buy and sell basic game keys.
2. [ ] The peer-to-peer resell system connects users for secure transfers.
3. [ ] Apple and Amazon gift cards are seamlessly supported as payment options.
4. [ ] Users can manage their shopping cart and track their detailed order status.

---

## Big User Story

**As a** User,
**I want to** securely buy, sell, and resell game keys using my preferred payment methods,
**So that** I can manage my transactions seamlessly in a single marketplace.
